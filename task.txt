Добрый день. Тема: проект "Создать тг бота"


Описание API-запросов

1. Авторизация пользователя через tg

Этот запрос используется для авторизации пользователя через tg. 
При входе с фронтенда передаются данные пользователя tg 
(имя, фамилия, ID, аватар). 
Если пользователь отсутствует в базе данных, 
он автоматически регистрируется. 
Сервер возвращает данные пользователя, 
включая список сделок и кошельков с балансом.

Запрос:

{

  "firstName": "string",

  "lastName": "string",

  "id": "number",

  "avatar": "string"

}

Ответ:

{

   "wallet": "string"

}

2. Получение списка игроков

Этот запрос возвращает список игроков или друзей.

Метод: GET

Параметры:

type (обязательный): Указывает тип данных (friends или players).

Ответ:

[

   {

        "fullName": "string",

        "league": "string",

        "coins": "number",

        "avatar": "string",

        "isFriend": "boolean"

   }

]

3. Добавление в друзья

Этот запрос используется для добавления пользователя в список друзей.

Метод: GET

Параметры:

id (обязательный): ID пользователя, которого нужно добавить в друзья.

Ответ:

{

  "status": "success" | "fail"

}
